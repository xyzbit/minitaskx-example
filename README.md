# minitaskx-example
minitaskx Use cases

# Usage
1. start minitaskx depend
```
make init
```

2. start minitaskx server
```
# run worker to execute task
make worker

# run scheduler to schedule task
make scheduler
```

3. use command-line tool
```
make build

./minitaskx -h
```
your can do:
   - create task
   - pause task
   - resume task
   - stop task

4. clean up the resources
make clean
